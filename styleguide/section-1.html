1 按鈕(Button)

按鈕支援：<a>、<button>、<input type="button">

1.1 預設 (Default)

.pure-button
<input type="button" class="pure-button" value="button"/>
<input type="button" class="pure-button-primary" value="button"/>
<input type="button" class="pure-button-success" value="button"/>
<input type="button" class="pure-button-error" value="button"/>
<input type="button" class="pure-button-warning" value="button"/>
<input type="button" class="pure-button-pink" value="button"/>

1.2 點擊(active)

.pure-button .active
<input type="button" class="pure-button active" value="button"/>
<input type="button" class="pure-button-primary active" value="button"/>
<input type="button" class="pure-button-success active" value="button"/>
<input type="button" class="pure-button-error active" value="button"/>
<input type="button" class="pure-button-warning active" value="button"/>
<input type="button" class="pure-button-pink active" value="button"/>

1.3 無法點擊(disabled)

.pure-button .disabled
<input type="button" class="pure-button disabled" value="button"/>
