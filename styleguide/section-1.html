1 按鈕

Button styles can be applied to anything with the .btn class applied. However, typically you'll want to apply these to only <a> and <button> elements for the best rendering.

1.1 狀態按鈕2

.button

Indicate that the button is the primary feature of this form.

<input type="button" class="btn-default" value="button"/>
<input type="button" class="btn-primary" value="button"/>
<input type="button" class="btn-waring" value="button"/>
